Instructions

Illustrator Files

Ready to use illustrator (.ai) files. Organized in a flexible (fabric) pattern and a rigid (wood) pattern



Inkscape SVG files

The SVG files are converted illustrator files for use with open source software, in this project they have been created and used wit Inkscape



3D Print Files

Some of the 3D objects need to be duplicated; each filename has (if it is required) a x2 or x4 indicating how many duplicates should be printed.
